# .impeccable.md

## Design Context

### Users
AI 에이전트(Claude Code 등)를 동시에 여러 개 운영하는 개발자. 멀티 터미널을 그리드로 배치하고, 각 패널에 캐릭터 이미지를 오버레이하여 에이전트를 시각적으로 구분한다. 작업 맥락: 장시간 모니터링, 빠른 패널 전환, 키보드 중심 워크플로우.

### Brand Personality
**Precise, Calm, Delightful**
- 새벽의 고요함 속에서 작업에 몰입하는 느낌
- 집중을 돕는 절제된 세련미 + 캐릭터 오버레이가 주는 즐거움의 균형
- Linear, Raycast처럼 불필요한 것을 덜어낸 프로덕트 감성

### Aesthetic Direction
- **Visual tone**: Cool & Natural. 포레스트 그린 계열의 자연스러운 다크 테마. AI 터미널 클론의 cyan glow를 탈피한 독자적 팔레트.
- **Color system**:
  - Background: `#111111` (neutral dark)
  - Surface: `#181818` (elevated surface)
  - Panel: `#141514` (terminal background)
  - Border: `#222322` (subtle separation)
  - Border light: `#2f302f` (elevated borders)
  - Text primary: `#e5e6e5` (high contrast)
  - Text secondary: `#cccdcc` (body text)
  - Text muted: `#707170` (labels, hints)
  - Text dim: `#4e4f4e` (disabled, decorative)
  - Accent: `#4ade80` (forest green — focus, active, interactive)
  - Accent glow: `rgba(74, 222, 128, 0.15)`
  - Accent hover: `rgba(74, 222, 128, 0.1)`
  - Accent selection: `rgba(74, 222, 128, 0.2)`
  - Danger: `#f87171` (warm red)
  - Success: `#86efac` (soft green)
  - Interactive bg: `#1e1f1e` (buttons, inputs)
- **Typography**: System monospace stack (`-apple-system`, `SF Mono`, `Menlo`)
- **Spacing**: Compact — 4px grid gaps, 3-8px padding. 터미널 공간 최대화 우선
- **Borders**: 1px solid, low-contrast. Active 상태에서만 accent color + subtle glow
- **Anti-references**: Cyan glow 터미널 (Warp/Hyper 클론), VS Code 복잡성, glassmorphism
- **References**: Linear (절제된 다크 UI), Raycast (키보드 중심 인터랙션), Ghostty (터미널의 개성)
- **App Icon**: 인디고 계열로 변경 예정 (현재 녹색 구체 → 새 팔레트에 맞게)

### Design Principles

1. **Terminal First** — 터미널 콘텐츠가 항상 최우선. UI 크롬은 최소화하고, 모든 여백은 터미널에 양보한다.
2. **Keyboard Native** — 마우스 없이도 모든 핵심 동작이 가능해야 한다. 시각적 피드백은 키보드 포커스 상태를 명확히 전달한다.
3. **Quiet Until Active** — 기본 상태는 눈에 띄지 않고, 인터랙션 시에만 accent color로 존재감을 드러낸다.
4. **Character as Identity** — 캐릭터 오버레이는 핵심 기능이자 tgrid의 차별점. 터미널을 가리지 않으면서도 에이전트의 존재감을 시각적으로 전달한다.
5. **Joyful Precision** — 절제된 인터페이스 안에서도 디테일이 즐거움을 준다. 부드러운 전환, 캐릭터의 미세한 반응, 프리셋 색상의 개성.
